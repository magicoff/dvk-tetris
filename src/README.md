# Tetris — Текстовый режим

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![JavaScript](https://img.shields.io/badge/JavaScript-Vanilla-yellow)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

Классическая игра Тетрис, реализованная полностью в текстовом режиме с использованием ASCII-графики. Игра работает в браузере и имитирует ретро-дисплей 80×24 символа.

## Особенности

- Чистый JavaScript без внешних зависимостей
- Поддержка полной системы управления клавиатурой и цифровой клавиатурой
- Ретро-стиль с эффектом свечения (glow)
- Система уровней и подсчёта очков
- Хранение рекорда в `localStorage`
- Поддержка мгновенного сброса (hard drop) и вращения с wall kick
- Реализация 7-bag системы для честной генерации фигур
- Адаптивный рендеринг на 80x24 символах
- Автоматическое падение фигур с увеличивающейся скоростью
- Система паузы с оверлеем
- Оптимизированный рендер без утечек памяти DOM

## Управление

- **Стрелки влево/вправо**: перемещение фигуры
- **Стрелка вверх**: поворот фигуры
- **Стрелка вниз**: мягкое падение (soft drop, +1 очко)
- **Пробел**: мгновенный сброс (hard drop, +2 очка за клетку)
- **P или ESC**: пауза
- **Enter или Пробел после Game Over**: рестарт
- **Цифра 0**: показать/скрыть подсказки управления
- **Цифра 1**: показать/скрыть следующую фигуру

## Технические детали

- Игровое поле: 10×20 ячеек
- Дисплей: 80×24 символа
- Шрифт: monospace (системный)
- Цветовая схема: зелёный неон на чёрном фоне
- Эффект свечения реализован через CSS-фильтр blur и дублированный слой
- Рендеринг оптимизирован: glow-слой переиспользует DOM-элемент

## Начисление очков

- **Мягкое падение (soft drop)**: +1 очко за клетку
- **Мгновенный сброс (hard drop)**: +2 очка за клетку
- **Очистка линий**: 100 / 300 / 500 / 800 × уровень (за 1 / 2 / 3 / 4 линии)
- **Повышение уровня**: каждые 10 очищенных линий
- **Увеличение скорости**: на 100 мс за уровень (минимум 100 мс)

## 7-bag система генерации фигур

Игра использует систему 7-bag для генерации фигур, что обеспечивает более честное и предсказуемое распределение тетромино по сравнению с полностью случайным выбором.

### Принцип работы:

1. Все семь типов фигур (I, J, L, O, S, T, Z) помещаются в "мешок"
2. Мешок перемешивается случайным образом
3. Фигуры достаются из мешка по одной
4. Когда мешок пуст, он снова заполняется всеми семью фигурами и перемешивается

### Преимущества:

- Гарантирует, что каждые семь фигур будут включать по одному экземпляру каждого типа
- Исключает ситуации с длительным отсутствием нужной фигуры
- Предотвращает появление слишком большого количества одинаковых фигур подряд
- Создаёт более сбалансированный и справедливый игровой процесс

Эта система используется в официальных версиях Tetris и считается стандартом для современных реализаций.

## Запуск

Просто откройте `index.html` в браузере. Сервер не требуется.

```bash
# Или используйте простой HTTP-сервер:
python3 -m http.server 8000
# Затем откройте: http://localhost:8000
```

## GitHub Pages

Проект автоматически деплоится на GitHub Pages при каждом пуше в ветку `main`.

### Настройка:

1. Перейдите в **Settings → Pages** вашего репозитория
2. В разделе **Build and deployment** выберите:
   - Source: **GitHub Actions**
3. Игра будет доступна по адресу: `https://<username>.github.io/<repo-name>/`
