# DVK Tetris

ASCII-версия Tetris в стиле терминала, написанная на чистых `HTML/CSS/JavaScript` без зависимостей.

## Запуск

1. Откройте `docs/index.html` в браузере.
2. Игра стартует автоматически после загрузки страницы.

## Управление

- `ArrowLeft` / `ArrowRight` - движение влево / вправо
- `ArrowUp` - поворот фигуры
- `ArrowDown` - мягкое падение
- `Space` - жесткий сброс (hard drop)
- `P` или `Esc` - пауза / продолжить
- `R` - перезапуск
- `0` - показать/скрыть текст подсказок справа
- `1` - показать/скрыть превью следующей фигуры
- После `Game Over`: `Enter` или `Space` для рестарта

## Особенности

- Поле `10x20`
- Рендер в текстовый дисплей `80x24`
- Генерация фигур по системе `7-bag`
- Рост скорости с уровнями
- Сохранение рекорда в `localStorage`

## Структура

- `docs/index.html` - точка входа
- `docs/style.css` - стили интерфейса
- `docs/app/app.js` - игровая логика
